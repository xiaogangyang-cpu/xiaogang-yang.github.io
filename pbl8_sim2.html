核心理论

01
活化能与能垒
反应所需跨越的能量势垒 $E_a$： $$E_a = E_{TS} - E_{reactants}$$ ⚠ $E_a\uparrow$ → 温度敏感性 $\uparrow$ ✓ $E_a\uparrow$ → 热稳定性 $\uparrow$
  • $E_a$ 越高，相同温度下降解越慢
  • 活化能由实验（Arrhenius图）测定
02
Arrhenius 公式
速率常数与温度关系： $$k = A\,e^{-E_a/(RT)}$$
  • $A$：指前因子（频率因子）
  • $R$：气体常数 8.314 J/mol·K
  • $T$：绝对温度 (K)
取对数：$\ln k = \ln A - \dfrac{E_a}{RT}$，以 $1/T$ 作图斜率为 $-E_a/R$。
03
Arrhenius 寿命外推
材料失效时间正比于： $$t_f \propto e^{E_a/RT} \quad \Rightarrow \quad \frac{t_2}{t_1} = e^{\frac{E_a}{R}\left(\frac{1}{T_2}-\frac{1}{T_1}\right)}$$ ✓ 高温加速实验外推室温寿命

加速因子 $AF = e^{E_a/R(1/T_{use}-1/T_{test})}$，典型值 50–200×

04
水分与添加剂对 Eₐ 的影响
综合有效活化能修正模型： $E_a^{eff} = E_a^{base} - \alpha W + \sum_i \Delta E_a^{(i)}$ ⚠ 水分（W ppm）→ 降低 Ea，加速降解 ✓ 添加剂 VC/FEC/TMSP → 提升 Ea，增强稳定性
  • VC：+8 kJ/mol，优先成膜消耗水分
  • FEC：+12 kJ/mol，富 LiF SEI，热稳定性强
  • TMSP：+15 kJ/mol，清除 HF/H₂O 最强效

工业标准：水含量 < 20 ppm（干燥间露点 −40°C）

05
SEI 膜动力学
✓ SEI 形成：保护负极，阻止进一步分解 ⚠ SEI 分解：产生气体，影响寿命 $r_{gas} = k_{gas} \cdot C_{elec}$ 气体产物：CO₂ H₂ C₂H₄

SEI 分解 $E_a$（稳定）≈ 70–90 kJ/mol；不稳定 SEI ≈ 40–60 kJ/mol

📘 理论背景 · Theoretical Background

1. Arrhenius 方程的推导与意义

1889年，Svante Arrhenius 基于实验数据提出速率常数与温度的定量关系：

k = A · exp(−Eₐ / RT)

两边取自然对数：ln k = ln A − Eₐ / (RT)。以 1/T 为横轴、ln k 为纵轴，可得一条直线，斜率为 −Eₐ / R，截距为 ln A。这是实验测定活化能的标准方法。

2. 反应速率对温度的敏感性

温度从 T 升高到 T+ΔT 时，速率变化倍率为：

k(T+ΔT)/k(T) = exp[Eₐ·ΔT / (R·T·(T+ΔT))]

室温下（T=298K），每升高 10°C，速率约增大 1.5–4 倍（取决于 Eₐ）。Eₐ = 80 kJ/mol 时，每升 10°C 速率约翻 2.7 倍。

3. LiPF₆ 水解机理与电解液稳定性

LiPF₆ 是锂离子电池中最广泛使用的导电盐，但对水极敏感。水解反应分步进行：

步骤1：LiPF₆ ⇌ LiF + PF₅
步骤2：PF₅ + H₂O → POF₃ + 2HF
总：LiPF₆ + H₂O → LiF + POF₃ + 2HF

生成的 HF（氢氟酸）腐蚀 LiCoO₂ 等正极材料（溶出 Co²⁺），破坏 SEI 膜完整性，导致容量衰减加速。该反应近似一级动力学，Eₐ ≈ 50–65 kJ/mol，活化能随水含量增加而降低。

4. SEI 膜形成与分解机制

固体电解质界面（SEI）膜在首次充电时形成，主要成分：

无机层：Li₂CO₃、LiF、Li₂O（内层，致密）
有机层：ROCO₂Li、(CH₂OCO₂Li)₂（外层，疏松）

SEI 分解反应（温度>60°C 显著）：

ROCO₂Li → Li₂CO₃ + C₂H₄↑ + CO₂↑ + H₂↑

稳定 SEI（富 LiF）的 Eₐ ≈ 80–100 kJ/mol；不稳定 SEI（富有机物）的 Eₐ ≈ 40–60 kJ/mol。

5. 工程案例：电池寿命预测标准

依据 IEC 62133 与 USABC 标准，通过高温加速老化实验预测室温寿命：

t₂ = t₁ · exp[Eₐ/R · (1/T₂ − 1/T₁)]

示例：Eₐ = 65 kJ/mol，在 60°C（333K）测试 200h → 预测 25°C（298K）寿命约 40,000 小时（≈4.6年）。加速因子 AF ≈ 200，与业内数据吻合。
