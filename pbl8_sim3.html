QUESTION 01 / 03 · 动力学原理
已知 LiPF₆ 在 60℃ 时的水解反应对水为二级、对LiPF₆为一级，经验速率常数 k = 0.0012 L²·mol⁻²·h⁻¹，初始浓度 C₀ (LiPF₆)= 1.0 mol/L，因水为痕量（≈0.01 mol/L且缓慢消耗），可近似用准二级模型针对LiPF₆进行简化计算： 1/C(t) − 1/C₀ = k′·t（其中k′已折算为x含水量下用LiPF₆表示的有效二级常数 0.00015 L·mol⁻¹·h⁻¹），经过t=10天 (240 h) 后，剩余 LiPF₆ 浓度约为多少？（提示：e不参与，二阶公式直接用）
✓ 正确！1/C − 1/1.0 = 0.00015 × 240 → 1/C = 1 + 0.036 → C ≈ 0.965 mol/L。这说明即使只损失3–5%，60℃下储存10天电解液就已出现明显性能风险，符合“微量分解即致命”的电池实际。
✗ 错误。代入公式：1/C − 1/1.0 = 0.00015 × 240 → 1/C = 1 + 0.036 → C ≈ 0.965 mol/L，选 A 为正确答案。
QUESTION 02 / 03 · Arrhenius 分析
利用 Arrhenius 方程分析：已知 k(25℃) = 0.0012 h⁻¹，k(60℃) = 0.0098 h⁻¹。 若要使 60℃ 下的速率常数降低至 0.0020 h⁻¹（约 1/5），有效温度需降低到约多少℃？ （已知活化能 Ea ≈ 75 kJ/mol，R = 8.314 J/mol·K）
✓ 正确！利用 Arrhenius 方程反推：当 k = 0.0020 时，ln(k) = ln(0.0020) ≈ -6.21，代入 ln A − Ea/(RT) 求解 T ≈ 315 K ≈ 42℃。热管理策略将温度从 60℃ 降至 42℃，速率降低约 5×。
✗ 错误。需通过 Arrhenius 方程：k = A·exp(−Ea/RT)，代入 Ea = 75 kJ/mol，求解目标温度。正确答案为约 42℃（A 选项）。

📖 机理参考手册
▶ 核心反应方程式
LiPF₆ ⇌ LiF + PF₅
PF₅ + H₂O → POF₃ + 2HF
HF + Li₂CO₃ (SEI) → 2LiF + H₂O + CO₂
SEI破损 → 更多LiPF₆暴露 → 自催化循环
▶ 一阶动力学
dC/dt = −k·C
解：C(t) = C₀·exp(−k·t)
线性化：ln(C) = ln(C₀) − k·t（lnC-t 直线）
▶ Arrhenius 方程
k = A·exp(−Ea/RT)
ln k = ln A − Ea/(R·T)
两点法：Ea = R·ln(k₂/k₁) / (1/T₁ − 1/T₂)
▶ 自催化机制特征
· 衰减速率随时间加速（非恒速）
· 存在加速拐点
· 产物（HF）作为催化剂促进水解
· 表现为"S形"或超指数衰减特征
